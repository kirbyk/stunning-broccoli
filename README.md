Stunning Broccoli
=================

![](http://www.healthforthewholeself.com/wp-content/uploads/2010/01/Broccoli.jpg)

Installation
------------

```bash
$ npm install -g gulp bower
$ npm install
$ bower install
```

Deployment
----------

#### Development

```bash
$ npm run build
$ npm start
$ browser http://localhost:8080
```
